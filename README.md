🎮 Scramblox

A Web-Based Word Scramble Game

Welcome to Scramblox, an interactive web-based word scramble game designed to make vocabulary learning fun and engaging. Players are challenged to rearrange scrambled letters to form meaningful words within a time limit.

This project demonstrates how simple web technologies can be used to build an engaging and educational gaming experience.

✨ Features

🔤 Word Scramble Gameplay
Players are given scrambled letters and must rearrange them to form the correct word.

⏱ Timed Challenges
Each round has a timer to encourage quick thinking.

📈 Score Tracking
Players earn points for correct answers.

🎯 Difficulty Levels
	•	Easy
	•	Medium

🎨 Modern UI Design
Clean interface designed using Figma concepts and modern web styling.

🔊 Interactive Sound Effects
Subtle UI sounds improve the gameplay experience.


⚙ How the Game Works

1️⃣ The game selects a random word from the word list.

2️⃣ The word is scrambled using a shuffle algorithm.

3️⃣ The scrambled word appears on the screen.

4️⃣ The player enters the correct word guess.

5️⃣ If the answer is correct:
	•	Score increases
	•	New word is generated

6️⃣ The game continues until the timer ends, after which the final score is displayed.

🧠 Algorithms Used

🔀 Fisher–Yates Shuffle Algorithm
Used to scramble letters randomly.

🎲 Random Word Selection
Randomly selects a word from the word database.

✔ Answer Validation Logic
Compares player input with the correct word.

📊 Score Calculation System
Updates the player’s score based on correct answers.
